

# Sistema de Pedidos - Lanchonete - Desafio Vai na Web

## Descrição
Este projeto apresenta a modelagem de um banco de dados para uma lanchonete que precisava organizar seus pedidos, substituindo o controle manual.

## Estrutura
O banco de dados é composto por:

- clientes
- produtos
- pedidos
- itens_pedido

## Justificativa da solução
A solução foi estruturada para resolver o problema de organização dos pedidos da lanchonete. Foram criadas tabelas para clientes, produtos, pedidos e itens do pedido, 
permitindo registrar de forma centralizada quem realizou o pedido, quais produtos foram solicitados, em que quantidade, modalidade de entrega e qual o valor total da compra.
